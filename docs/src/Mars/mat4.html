mat4

Class designed for performing operations on 4x4 float matrices. Uses SSE2 for vectorization.

Constructors

mat4() - sets all values of matrix to 0
mat4(f32 val) - sets all values along the main diagonal to val. Call like mat4(1.f) to create an identity matrix.
mat4(__m128 a, __m128 b, __m128 c, __m128 d) - sets rows of values of matrix to a, b, c, and d respectively

Functions

mat4 LookAtLH(vec3 position, vec3 target, vec3 up)

Returns a view matrix specified by input parameters. Generally used to find what a camera is looking at.

mat4 PerspectiveFovLH(f32 angle, f32 aspect_ratio, f32 _near, f32 _far)

Returns a perspective matrix specified by input parameters. angle is the FOV of the camera. Needed in creation of a camera for 3D games.

mat4 Transpose(mat4 matrix)

Returns the transpose of the input matrix. Sometimes needed to send matrix data to the GPU depending on the rendering API.

mat4 Scale(vec3 scale)

Scales the calling matrix by the scale value specified by the input vec3.

mat4 Rotate(vec3 axis, f32 angle)

Returns a rotation matrix by rotating the calling matrix by amount angle along arbitrary axis.

mat4 Translate(vec3 translate)

Returns a translation matrix using input vec3.

Operators

mat4 operator*(mat4 left, const mat4 right)

Performs basic matrix multiplication operation and returns the result.

Data

__m128 data[4] - array of 4 __m128 values for 16 SIMD f32 values in total
