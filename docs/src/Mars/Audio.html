Audio

The audio engine for the Mars Engine powered by the XAudio2 library. Currently only capable of playing 2D audio files so no 3D audio effects or editing options such as reverb, pitch, looping, etc.

Functions

void Init()

This currently gets called only directly before playing an audio file. This could probably be called only once in system startup, however not much testing has been done with the audio engine. Returns void.

void PlayAudio(char* file_name)

This is going to play an audio file with no options for looping, pausing, stopping. Currently only tested with .wav files... Returns void.
