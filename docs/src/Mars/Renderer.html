Renderer

Houses all of the logic for initializing DX11 & DX12. Also contains all of the logic necessary for putting images onto the drawing surface.

Functions

void InitDX12()

Has not been implemented yet...

void InitDX11()

Initializes data needed to start up a DX11 instance. Returns void.

void InitScene

Gets scene ready to draw graphics onto the screen. Sets up important information not necessary for initialization of the renderer. Returns void.

void UpdateRenderer()

Update loop for the renderer. Getting the world prepped before each draw call. Returns void.

void Draw()

Draw call for the renderer. Sets background color, sends new data to the GPU, draws index data, and swaps the buffers in the swap_chain. Returns void.

void TerminateDX11()

Frees all of the memory allocated by DX11 objects during the process of running the application. Returns void.
