Application

The Application class houses the main loop of the program and is also in charge of system startup/shutdown.

Functions

void GameStartup()

This is a virtual function meant to be implemented by the game. Use this function to set game specific data (ie. entity data, game rules) for startup. Returns void.

Application* CreateApplication()

To be implemented by the game. Use this to set input bindings for the game using SetInput. Also use this to set system-wide globals such as height and width of the game window. Returns pointer to new Application.
