MVector

C++ style vector for the Mars Engine.

Constructors

MVector<T>() - allocates enough space for 10 items to be entered into vector
MVector<T>(s32 _size) - allocates enough space for _size + 1 items to be entered into vector
MVector<T>(const MVector<T>& other) - copy constructor

Destructors

~MVector() - cleans up memory left behind by vector.

Functions

void Resize(s32 sz)

Sets the new size for the vector. Returns void.

s32 Size() const

Returns the number of elements in the vector.

T* begin()

Returns a pointer to the first element in the vector. Necessary for using range-for.

T* end()

Returns a pointer to the last element in the vector. Necessary for using range-for.

bool Find(T t)

Returns true if t is an element in the vector. Returns false if not.

s32 FindIndex(T t)

Returns index of t in the vector. If element does not exist in vector, -1 is returned.

void PushBack(T val)

Adds val to the end of the vector.

void Remove(s32 index)

Removes the value held in the vector with the index provided by function parameter.

Operators

T& MVector<T>::operator[](s32 index)

Allows use of [] operator to treat vector as a C-style array.

Data

T* data - array holding the data of the vector
s32 capacity - the temporary maximum capacity of the vector
s32 size - the number of elements in the vector
