vec3

Class designed to handle operations involving 3-dimensional vectors. Uses SSE2 for vectorization.

Constructors

vec3() - sets all elements of the vector to 0
vec3(f32 val) - set first 3 elements of the vector to val
vec3(f32 _x, f32 _y, f32 _z)
vec3(__m128 v) - forms vector from __m128 input

Functions

f32 x()

Returns the x value of the vector.

f32 y()

Returns the y value of the vector.

f32 z()

Returns the z value of the vector.

f32 Dot(vec3 a, vec3 b)

Returns inner product of a and b.

f32 Length()

Returns the magnitude of the vector as a f32.

vec3 Normalize(vec3 v)

Returns v as a normal vector (vector with Length() == 1).

vec3 Cross(vec3 a, vec3 b)

Returns the cross product of a and b. Remember the cross product is non-commutative so Cross(v1, v2) != Cross(v2, v1).

vec3 yzx()

Swaps the elements of the vector. Commonly known as a swizzle operation.

vec3 zxy()

Swaps the elements of the vector. Commonly known as a swizzle operation.

f32 Sum()

Returns the sum of all of the elements of the vector.

Operators

vec3 operator+(vec3 a, vec3 b)

Adds the elements of a and b and returns the result.

vec3 operator-(vec3 a, vec3 b)

Subtracts the elements of a and b and returns the result.

vec3 operator*(vec3 a, vec3 b)

Multiplies the elements of a and b and returns the result.

vec3 operator*(vec3 a, f32 b)

Multiplies the elements of a with f32 b and returns the result.

vec3 operator*(f32 a, vec3 b)

Multiplies the elements of b with f32 a and returns the result.

vec3 operator/(vec3 a, f32 b)

Divides the elements of a by f32 b and returns the result.

std::ostream& operator<<(std::ostream& os, vec3 a)

Gives the ability to print this class out using std::cout or any of the macros set by Core.h

Data

__m128 data - SSE2 data type with a size of 16 bytes used for SIMD. The size is important to note because most other libraries will expect a size of 12 (3 elements * 4 bytes per f32).

Other

Macros

SHUFFLE3(V, X,Y,Z) - helper macro for defining swizzling functions.
