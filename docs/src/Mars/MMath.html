MMath

Math library for the Mars Engine.

Functions

f32 ToRadians(f32 val)

Converts val from degrees to radians. Returns the outcome of the conversion.

f32 ToDegrees(f32 val)

Converts val from radians to degrees. Returns the outcome of the conversion.

f32 Min(f32 a, f32 b)

Returns the smaller of the two floating point input values.

f32 Max(f32 a, f32 b)

Returns the larger of the two floating point input values.

f32 Floor(f32 val)

Rounds val down. Returns the outcome of the calculation.

f32 Clamp(f32 bottom, f32 top, f32 val)

Keeps val within the bounds set by bottom and top. Returns bottom if val is less than bottom, top is val is greater than top, or val if neither is true.

f32 RandFloat(f32 min, f32 max)

Returns a random f32 with the value being bound by min and max.

s32 Min(s32 a, s32 b)

Returns the smaller of the two integer input values.

s32 Max(s32 a, s32 b)

Returns the larger of the two integer input values.

u32 Abs(s32 a)

Returns the absolute value of the input as a u32.

s32 Clamp(s32 bottom, s32 top, s32 val)

Keeps val within the bounds set by bottom and top. Returns bottom if val is less than bottom, top is val is greater than top, or val if neither is true.

s32 RandInt(s32 min, s32 max)

Returns a random s32 with the value being bound by min and max.
