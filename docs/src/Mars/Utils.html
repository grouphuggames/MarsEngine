Utils

Misc extra utilies such as a timer

Functions

void StartTimer(TimerInfo& info)

Starts a timer using info struct provided as input parameter. Sets info.timer_start to the time this function is called. info.time_scale must be set before calling this function.

void StopTimer(TimerInfo& info)

Stops the timer using info struct provided as input parameter. Sets info.timer_stop to the time this function is called and sets info.time_delta to the elapsed time since StartTimer was called.

void ShowFPSCounter(bool active)

Sets whether or not the game's framerate should be shown in the debug console. Should be placed in a loop to see it constantly updating.
