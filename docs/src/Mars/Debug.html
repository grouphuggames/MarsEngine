Debug

Defines information for creating debug console and logic for the hot reload system.

Functions

void CreateWin32DebugConsole()

Creates a console to put debug messages into. Returns void.

u32 string_hash(const char* data, size_t data_length, const char* salt, u32 salt_length)

Takes string and hashes it into an unsigned 32 bit integer. Used to create keys for HRVars data structure. Returns data as a (hopefully) unique u32.

static void AddHot(u32 name_hash, s32* var_ptr)

Adds signed integer to list of variables for hot reloading (hot_reload_vars.ints). Returns void.

static void AddHot(u32 name_hash, f32* var_ptr)

Adds float to list of variables for hot reloading (hot_reload_vars.floats). Returns void.

static void AddHot(u32 name_hash, bool* var_ptr)

Adds bool to list of variables for hot reloading (hot_reload_vars.bools). Returns void.

static void AddHot(u32 name_hash, vec3* var_ptr)

Adds vec3 to list of variables for hot reloading (hot_reload_vars.vec3s). Returns void.

void HotReload()

Changes the values of all variables in hot_reload_vars from values in memory to values specified in HotReload.txt. Returns void.

HRVars

MVector<std::tuple<u32, s32*>> ints;
MVector<std::tuple<u32, f32*>> floats;

MVector<std::tuple<u32, vec3*>> vec3s;

Other

Defines

SALT - Salt to be used by string_hash. Default value is "MARS".
SIZE_OF_SALT - Must be equal to the number of characters in SALT including the null terminating character. Default value is 5.

Macros

ADDHOT(var) - calls AddHot function according to the data type of var.
