Core

This defines the core of the engine and contains both the engine level and game level debugging messages as well as global data to be accessed by all other areas of the engine or game.

Functions

void PrintToLog(T&&... t)

Takes any data type with an ostream operator overload and prints it out to the debug console. Can be any number of mismatched data types. Returns void.

Macros

MARS_CORE_INFO_EXTRA(...) - prints out debug info to debug console in green font from the engine with file and line number attached
