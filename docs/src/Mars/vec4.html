vec4

Class designed to handle operations involving 4-dimensional vectors. Uses SSE2 for vectorization.

Constructors

vec4() - sets all elements of the vector to 0
vec4(f32 val) - set all elements of the vector to val
vec4(f32 _x, f32 _y, f32 _z, f32 _w)

Functions

f32 x()

Returns the x value of the vector.

f32 y()

Returns the y value of the vector.

f32 z()

Returns the z value of the vector.

f32 w()

Returns the w value of the vector.

f32 Dot(vec4 a, vec4 b)

Returns inner product of a and b.

f32 Length()

Returns the magnitude of the vector as a f32.

f32 Sum()

Returns the sum of all of the elements of the vector.

Operators

vec4 operator+(vec4 a, vec4 b)

Adds the elements of a and b and returns the result.

vec4 operator+=(vec4& a, vec4 b)

Adds the elements of a and b and stores the result in a.

vec4 operator-(vec4 a, vec4 b)

Subtracts the elements of a and b and returns the result.

vec4 operator*(vec4 a, vec4 b)

Multiplies the elements of a and b and returns the result.

vec4 operator*(vec4 a, f32 b)

Multiplies the elements of a with f32 b and returns the result.

vec4 operator*(f32 a, vec4 b)

Multiplies the elements of b with f32 a and returns the result.

vec4 operator/(vec4 a, f32 b)

Divides the elements of a by f32 b and returns the result.

vec4 operator/=(vec4& a, f32 b)

Divides the elements of a by f32 b and stores the result in a.

std::ostream& operator<<(std::ostream& os, vec4 a)

Gives the ability to print this class out using std::cout or any of the macros set by Core.h

Data

__m128 data - SSE2 data type with a size of 16 bytes used for SIMD.
