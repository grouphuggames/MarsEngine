Input

Defines logic necessary for handling user input.

Functions

void SetInput(MARS_KEY input, InputFunc func)

Binds func to the keyoard button specified by input. Returns void.

void SetInput(MARS_MOUSE input, InputFunc func)

Binds func to the mouse button specified by input. Returns void.

LRESULT CALLBACK ProcessInput(HWND hwnd, UINT msg, WPARAM wParam, LPARAM lParam)

Handles all of the keyboard and mouse button inputs. If red 'X' or ESC key are triggered, the application will close. If binding has not been set by SetInput, an error message will be put out to the debug console. Returns LRESULT.

Data

std::unordered_map<s32, InputFunc> InputFuncs - This map uses MARS_KEY and MARS_MOUSE as its key value with the function pointer as the value. Used to store input bindings.

Other

Aliases

InputFunc = void(*)(void) - alias for function pointer

Enums

MARS_KEY - renames win32 virtual key codes for the keyboard to be more uniformly readable
MARS_MOUSE - renames win32 virtual key codes for the mouse to be more uniformly readable
